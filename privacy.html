Purpose
The purpose of this third-party pixel is to collect data from the performance API of JavaScript with the aim of building a model that estimates the amount of CO2 generated in online ads. Specifically, we gather the following information:
Data collection and processing
  • Number of clicks on ads: This data is used to estimate the CO2 generated per each click on the ad.
  • Performance data: We gather performance data to retrieve duration events, enabling us to compute more accurate CO2 consumption estimations.
  • performance.getEntries: This information is essential for obtaining a comprehensive dataset that influences CO2 generation estimations.
  • Navigator data: Understanding the user's platform helps us report CO2 usage based on the type of device or system being used.
  • Screen data: We consider the screen size to factor into our estimations.
  • Size of the ad iframe: The ad size contributes to the overall estimation of CO2 emissions.
  • Parent URL: Knowing the webpage where the ad is displayed helps us understand its influence on CO2 generation.
Data Access
We access the data by inspecting the HTML of the ads and analyze the requests and responses the ad performs to the different pixels and tags that serve to load the ad's content and track events with third-parties.
Data Storage
The collected information is also stored in a secure backend server located at Amazon Web Servers (AWS). Access to the server is limited to the team working on this project and is protected by user and password credentials.
